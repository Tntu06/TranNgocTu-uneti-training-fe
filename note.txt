.Class định danh css của class
#Id định danh css của id

Cách đặt tên
-Cú pháp: chấp nhận chữ cái thường, hoa , gạch ngang, gạch dưới
Không bắt đầu bằng số
Không chấp nhận ký tự đặc biệt

-Lưu ý: 
+ Đặt class có ý nghĩa(Hạn chế viết tắt)
+ Nên sử dụng đặt tên Tiếng Anh
+ Nên sử dụng quy tắc BEM( Blocks Element Modify)

Phân biệt sự khác nhau của class với id

I. Class
- Trong 1 thẻ HTML có nhiều class
- Độ ưu tiên của class thấp hơn ID
- Class tốt hơn cho việc kế thừa css

II. ID
- Trong html chỉ có 1 ID
- Độ ưu tiên ID lớn hơn class
- ID hạn chế trong việc kế thừa css

Khi nào dùng Class/id

Class dùng phổ biến trong các th
Id dùng khi muốn độ ưu tiên cao hơn class
ID Hỗ trợ JavaScript DOM
ID sử dụng cho các thành phần ít lặp lại

Selector
1.Chọn tất cả thẻ đó
p { }
div { }
ul { }
2.Phần tử có class=""
.class { }
3.ID selector id=""
#id{ }
4.class / id (KHÔNG dấu cách)
.box.active { }
div#main { }
5.Group selector
h1, h2, h3 { }
6.selector (dấu cách) p nằm bên trong div (mọi cấp)
div p { }
7.Child selector (>)
div > p { } #p con trực tiếp của div
8.Adjacent sibling (+)
p + ul { }
#ul ngay sau p
#Chỉ 1 phần tử
9.General sibling (~)
p ~ ul { }
#TẤT CẢ ul
#Sau p
#Cùng cha
